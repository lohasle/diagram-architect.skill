Features

Every generated diagram is a single, self-contained HTML file — open it, share it, no server needed.

🎨

Corporate navy, Dark Mode slate, Minimal grayscale, Tech purple-cyan, Warm amber — switchable at runtime without re-generating.

Zoom 25%–300% with mouse wheel or buttons, drag to pan, keyboard shortcuts. Works on desktop and mobile touch.

Installation

Get started in 30 seconds

Two ways to install — pick whichever fits your workflow.

Download and place the packaged .skill file in your skills directory:

# 1. Download the latest release
curl -L https://github.com/lohasle/diagram-architect.skill/releases/latest/download/diagram-architect.skill \
  -o ~/.claude/skills/diagram-architect.skill

# 2. Claude Code auto-discovers it on next launch
# No further configuration needed

Clone the source directly into your skills directory:

# Clone into skills directory as 'diagram-architect'
git clone https://github.com/lohasle/diagram-architect.skill.git \
  ~/.claude/skills/diagram-architect

# Update later with:
cd ~/.claude/skills/diagram-architect && git pull

Confirm the skill is installed:

# Check the skills directory
ls ~/.claude/skills/
# → diagram-architect/  (or diagram-architect.skill)

# In Claude Code, type /skills to see active skills
# diagram-architect should appear in the list

For D2/PlantUML rendering via render.py:

pip install requests
